## Main finding

The host start path is a UI callback followed by two best-effort broadcasts, not a distributed transaction. Roster and connection state can change while loading, host state becomes active before network publication, and clients can accept START_GAME or SYNC independently without one shared start ID, run session, epoch, acknowledgement or duplicate policy.
