## Finding

`runLoadingSteps()` has no generation or cancellation contract. Solo and host start functions resume after multiple yields and commit using retained inputs. GameCanvas then builds retained world resources once from the first snapshot, so a later stale or overlapping bootstrap can replace authoritative state without replacing world topology.
