## Source-backed finding

```txt
HUDOverlay screen guard:
  not PLAYING or COMPLETED -> null

HUDOverlay PLAYING branch:
  SettingsOverlay
  FrameDebugPanel
  no Minimap
  no objective/status panel

HUDOverlay COMPLETED branch:
  completed status panels
  Minimap
  SettingsOverlay
  FrameDebugPanel

GameCanvas each RAF:
  resolve runtime-minimap by DOM id
  call drawMinimapFrame

Minimap missing:
  drawMinimapFrame returns immediately
```

The minimap kit is therefore implemented and called but not reachable during active play. It becomes mountable only after the run transitions to `COMPLETED`.
